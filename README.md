## Getting Started

### Installing

1. Clone the repo:

```bash
git@github.com:Mare94/morten-sandstedt-js-framework-ca.git
```

2. Install the dependencies:

```
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run start
```
